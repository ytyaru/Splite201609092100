# このソフトウェアについて #

Splite201609092100は私個人が学習目的で作成したソフトウェアです。

Gdk::Pixbufクラスをもちいてスプライトアニメーションします。

# 開発環境 #

* Windows XP Pro SP3 32bit
* VC++ 2010 Express
    * C++/Win32

## ライブラリ ##

* [Gtkmm 2.22](http://ftp.gnome.org/pub/GNOME/binaries/win32/gtkmm/2.22/gtkmm-win32-devel-2.22.0-2.exe)
    * GTKmm 2.4
        * GTK+ 2.22.0
    * gdkbmm 2.4
        * gdk-pixbuf 2.0
    * glibmm 2.4
        * glib 2.0
    * cairomm 1.0
        * cairo 1.10.0
    * pangomm 1.4
        * pango 1.28.3
